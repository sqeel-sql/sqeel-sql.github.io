# sqeel.github.io

Landing page for [kryptic-sh/sqeel](https://github.com/kryptic-sh/sqeel).

Static HTML. No build step. Deploys via GitHub Pages from `main`.

## Deploy

1. Create repo `kryptic-sh/sqeel.kryptic.sh` (org site) — or push to `gh-pages` branch of `kryptic-sh/sqeel` (project site).
2. Settings → Pages → Branch: `main`, Folder: `/ (root)`.
3. Live at `https://sqeel.kryptic.sh/`.
