# sqeel.github.io

Landing page for [sqeel-sql/sqeel](https://github.com/sqeel-sql/sqeel).

Static HTML. No build step. Deploys via GitHub Pages from `main`.

## Deploy

1. Create repo `sqeel-sql/sqeel-sql.github.io` (org site) — or push to `gh-pages` branch of `sqeel-sql/sqeel` (project site).
2. Settings → Pages → Branch: `main`, Folder: `/ (root)`.
3. Live at `https://sqeel-sql.github.io/`.
